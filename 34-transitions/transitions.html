Transitions

  1. The transition property is a 'shorthand property'. It includes:
    1. transition-duration
    2. transition-timing-funcction
    3. transition-delay
  2. The difference between a timing function, delay, and duration:
    1. timing function is the speed.
    2. delay asks the element to move after 2s or so on of the first element's movement.
    3. duration the time on how long is the movement.
  3. These are some of animatable CSS properties:
    1. background-color
    2. border
    3. padding
    4. background-size
    5. heigth
    6. opacity
    7. font
    8. margin
    9. vertical-align
    10. visibility
  4. Yes. We can select more than one property to transition at a time by using the shorthand property: transition.
  5. We can define or set the desired transition state by using transition-timing-funcction.
  6. Transitions are useful for decorating or animating element.
  7. Transitions become a problem when we want to know where does the animation starts and ends.
  8. Transitions don't work reliably with the auto keyword being one of the end states because auto does not have unit.
  9. We can stagger transitions by using transition-delay.
