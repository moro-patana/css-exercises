All the small things

Small properties

  1. Cursor is an indicator helps user to point on which element they want instead of clicking. It is a pointer.
  2. Pointer-events is an indicator shows that the element is target.
  3. Outline is the ligne outside of the box. It does not include the width of the box.
  4. Filter is a kind of transparent material that absorbs light of some colors. It is used to change color.
  5. Appearance is used to override an element.
  6. Object-fit is a property to fit an element in a container.
  7. Scroll-snap-align helps the user to snap scrolling through the document.

Pseudo-class selectors

  1. :focus: targets focused element by using keyboard. e.g: tab
  2. :focus-within: selector reaches a parent element from a child. It also helps the users to optimize code.
  3. :visited: applies once the link which has been visited by the user.
  4. :active: is used to select an element in its activated state. What are the difference between :active and :focus?
  5. :target: is used for the element has an id.
  6. :disabled: represents the element which can not be selected.
  7. :empty: shows that the element has no children inside.
  8. :not(): a negation pseudo element to shows that the element is not match a list of selector.
  9. :matches(): selects the element that matches in a selector list.
  10. ::first-letter: selects the first letter of element.
  11. @support: is a rule which tests whether the user agent support property: value pair.
  12. attr(): abreviation of attribute we can use to retrieve hrf or url and so on of the element.

    2. attr(src url)
    3. data-count
