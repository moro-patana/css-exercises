Flexbox

  1. The order of property values in the shorthand flex property. i.e. Givenâ€¦ flex: 2 1 auto
    1. 2 1 represents the flex-grow.
    2. auto represents the width and height properties.
  2. Before we can use align-content, we need:
    1. flex-wrap: wrap
    2. flex-flow: column wrap or row wrap
  3. Difference between align-content and align-items:
    1. align-content manages the space between the wrap flex items.
    2. align-items controls the alignement of items on the cross axis
  4. The flex item's align-self overrides the other.
  5. The other commonly-used box model property similar to flex-basis is min-width
  6. There are two ways to change the ordering of flex items within a flex container. One is done on each flex item by using the order property. The other can change the whole order of the flex items at the container level. It is flex-direction: row-reverse or column-reverse.
  7. when we want some space between each of our three flex items in a flex column, we can use:
    1. display: flex
    2. justify-content: space between
