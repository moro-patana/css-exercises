Dimensions

The default values of:

  • width: auto
  • height: auto
  • min-width: 0
  • max-width: 0
  • min-height: 0
  • max-heigth: 0

How do they work?

The default values do not change whether it is block-like or inline-like element.

Comparaisons

  • If the min-width value is greater than the width, the width value of the element increases.
  • If the max-width value is greater than the width, the heigth value of the element increases.
  • If a min-width value is greater than a max-width value, the width of the element extends.
  • If it by default, the tables automatically sized by the cell contents.
  • The implied width of a div is width.
  • The deal with height and percentage values is that you have to apply the percentage to the parent element.
