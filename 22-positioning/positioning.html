Positioning

The default value for position is static.

  1. The other properties we need to supply when using any non-default position value are:
    1. margin
    2. width
    3. height
  2. The inferred display value of an element that has been positioned is: inherit.
  3. One of the non-default values for position still maintains a box in the flow of the document. It is the position: "relative";
    1. fixed positioning does not occupy any specific space.
    2. sticky positioning depends on the user's scroll position.
  5. When positioning an element absolutely, we can lock the element's coordinates by using arrow key.
  6. The absolute positioned are useful for for freestanding:
    1. tables.
    2. text buttons.
    3. logo images.
