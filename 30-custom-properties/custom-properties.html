Custom Properties

  1. when we define Custom Property's name, we have to use:
    1. Two dashes (--)
  2. We can not use hashes and slashes.
  3. We can assign number to a new Custom Property.
  4. It is useful to create Custom Properties using the :root or html selector when we want all of the element child inherit their parents values. However, we change anything if we set all the properties in the :root selector.
  5. After we've defined a new Custom Property, can recall or use it with Var() function.
  6. We can override or change a Custom Property by using:
    1. Media query
    2. Var() function
    3. Calc() function
  7. Custom Properties are useful because:
    1. Help us not to repeat ourselves when we write the code.
    2. Inherit their values from their parents.
  8. The only aspect (or part) of CSS syntax that we can use Custom Properties for is declaration block.
