Custom Properties

  1. when we define Custom Property's name, we have to use:
    1. Two dashes (--)
    2. Underscore
    3. Number
  2. We can not use spaces.
  3. We can assign to a new Custom Property by using:
    1. number
    2. text
    3. keyword
  4. It is useful to create Custom Properties using the :root or html selector when we want all of the element child inherit their parents values. However, we can not change anything if we set all the properties in the :root selector.
  5. After we've defined a new Custom Property, we can recall or use it with Var() function.
  6. We can override or change a Custom Property by:
    1. changing the value
    2. changing selector
    3. using media query
  7. Custom Properties are useful because:
    1. Help us not to repeat ourselves when we write the code.
    2. Write less code
    3. 1 point of control
    4. Inherit their values from their parents.
  8. The only aspect (or part) of CSS syntax that we can use Custom Properties for is declaration block (the Property and value pair).
