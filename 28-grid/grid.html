CSS Grid

  • By default, grid has one column.
  • The difference between Flexbox and Grid
    • Flexbox is designed for one dimensional layout: row or column.
    • Grid is designed for both: row and column at the same time.
  • I would use Flexbox because it controls all the child elements and adjusts the number of items in each row.
  • I would use Grid because we can use row and column at the same time.
  • A more concise way to write out the following: grid-template-columns: 10rem 10rem 10rem 10rem; is:
    • grid-template-columns: 10rem 1fr:
  • we can specify a minimum height for a grid row, but still allow the row to increase in height if its content is too tall by using:
    • grid-template-row: minmax;
