Keyframe animations

  1. Keyframe animation starts with @keyframes
  2. Keyframes code is different from the rest of CSS because:
    1. it has curly brackets.
    2. we have to specify where to start and end the animation.
    3. we can have many selector in one animation.
  3. We use a keyframe animation by using those properties: animation-name, animation-duration.
  4. These are the keywords that keyframe animations have but that aren't available in transitions:
    1. alternate
    2. infinite
    3. reverse
    4. alternate-reverse
    5. paused
    6. running
  5. We can animate as many animatable properties at once:
  6. These are animatable properties: background-color, font, padding, border, margin, opacity, visibility, word-spacing, width, vertical-align, text-decoration, order, color, z-index, flex, gap, height, box-shadow, clip, grid, text-indent.
