Shadows

  1. Difference between box-shadow and text-shadow:
    1. we can apply thickness in box-shadow, but in text-shadow we can not.
    2. inset does not work in text-shadow.
    3. in box-shadow, clip-path cut off any shadow that would extend below it.
  2. If you wanted to create a hard-edge shadow, one that looks like a solid border, you would need these values: x y thickness blur color.
  3. Yes we can create a shadow on a circular element if we set x and y values to 0, but with polygone we can not.
  4. Yes we can create multiple shaddow on the same element.
