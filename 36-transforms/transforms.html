Transforms

  1. These are the functions we can use in a trasform's values:
    1. Transform: scale()
    2. Transform: rotate()
    3. Transform: translate()
    4. Transform: skew()
    5. Transform: matrix()
    6. Transform: perpective()
    1. We use separate functions, in order, and as values, e.g. transform: scale(0.5) rotate(45deg) because the order is the most important.
    2. We could have had separate properties like: transform-scale: 0.5;, or transform-rotate: 45deg so that they will not delete each other.
  3. Often, we want to move the central pin or axis of the transform away from its default position. We can use transform-origin property to set that.
  4. It is preferable to use transform: translate() rather than position and its associated properties because in position you have to set position's values and left or right too.
    1. When an element is translated / transformed, the flow of the document does not change.
    2. Transform is similar to position property.
