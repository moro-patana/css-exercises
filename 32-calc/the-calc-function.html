The Calc function

  1. calc() function can support: number
  2. We can control the mathematical order of operation (PEDMAS / BODMAS) in the function by using:
    1. calc(var(2+3) * 4)
  3. Yes we can use a calc() function to convert one type of unit to another type of unit
  4. If we want to convert a unitless number to a unit value, what we can do is to add the unit as a string.
  5. The plus and minus operators inside the function need spaces around them because the operator can not resolve the problem if do not put spaces around them.
  6. We can convert a value into a negative value if we multiply it by negative value.
  7. Yes, calc() function works with colour values but we have to use rgb().
