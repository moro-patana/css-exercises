Inheritance

What do we mean by CSS inheritance?

It means the child element inherits the property value by default.

If you did not specify which element do you want to style, all of the element child will inherit the color or background-position, etc.

Which properties have values that are inherited by default?

These are the properties have values that are inherited by default:

  • border-spacing
  • color
  • font-family
  • font-size
  • font-style
  • font-variant
  • font-weight
  • font
  • letter-spacing
  • line-height
  • text-align
  • text-indent
  • text-transform
  • visibility
  • white-space
  • word-spacing

The difference between the `currentColor` and `inherit` keywords when used with colour values

Currentcolor is the color of the current text.

Inherit means the color of the current text is applied to the element child.

What are those keywords values do?

  1. revert: go back to the initial value of the property.
  2. initial: applies the default value of an element.
  3. unset: resets a property to its inherited value.

There are dangers or unintended side-effects when using those keywords:

revert and unset do not swipe out syles and allow inheritance.

inherit does not apply in an input element.

initial is unexpected on inherited properties.

These are the possibilities with using an ancestor or universal selector, combined with the `inherit` value:

  1. Universal selector applies style properties to all individual element.
  2. Inherited style properties can replaced by universal selector.
