Pseudo-element

  1. For a pseudo-element to work, the special property sets for the element is content
  2. We can use
    1. counter
  3. The various pseudo-element are:
    1. ::before
    2. ::after
    3. ::first-letter
    4. ::first-line
    5. ::section
  4. We use pseudo-element to specify where do we want to put the style.

Counters

  1. In CSS we use the following declaration to count:
    1. content: "[" counter(name) "] == [" counter(name, style or string) "]"
  2. We reset a counter by using this property with value: reset-counter: value;
