Generated content, counters, and lists

Pseudo-elements

For a pseudo-element to work, the special property sets for the element is content

The various pseudo-element are: ::after, ::before, ::first-letter, ::first-line, ::section

We use pseudo-element to specify where do we want to put the style.

Counters

In CSS we use the following declaration to count:
content: "[" counter(name) "] == [" counter(name,style or string) "]"

We reset a counter by using this property whit value:
reset-counter: value;
