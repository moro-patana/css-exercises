Floats and block formating contexts

Floats

  1. Default values:
    1. float: none
    2. clear: none
  2. The possible values for clear are:
    1. left
    2. right
    3. both
  3. The rule associated with a floated item in terms of source order is that the float item must go before.
  4. Yes. I think we can use float to another element besides an image.
  5. Yes. We can use pseudo element to create the two-columns layout.

BFC (Block formatting context) is a set of rule how the box interract to another and ceates private container.

  1. We can 'clear' a float by creating a block formatting context on the parent, this is achieved by putting a certain set of properties on the container. These are the properties:
    1. float: none
    2. overflow: visible
    3. display: inline-block
    4. position: not static or relative
