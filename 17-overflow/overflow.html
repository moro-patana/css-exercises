Overflow and Clipping

Default values

  • overflow: visible
  • clip: auto
  • clip-path: auto
  • text-overflow: hidden

Why would we use these properties? What makes them useful?

  • overflow: helps the users how to make the context fit on the container.
  • clip: it hides the rest of the context out of the countainer.
  • clip-path: specifies what element do you want to display.
  • text-overflow: deals with the overflowing element's box.

The other properties need to be set before these properties can work effectively:

  • overflow: width, heigth, border
  • clip: position
  • clip-path: margin
  • text-overflow: white-space, overflow

General questions

  1. The browser's default way to handle contents that are larger than its container is clip.
  2. An Ellipsis is the extra words replace by three dotes. An Ellipse is a shape function such as circle, rectangle.
  3. Text overflow has difference from the other properties because it helps the user when the text run out of the container.
